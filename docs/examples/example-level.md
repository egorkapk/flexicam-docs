# FlexiCam example level

Demo map that provides examples of [`Camera Modes`](../camera-mode.md) and how to use it with [`Camera Mode Trigger`](../camera-mode-trigger.md) and player input

## Interactive platforms

You can stand on

- `Green platform` to open documentation about example topic
- `Blue platform` to open example Blueprint
